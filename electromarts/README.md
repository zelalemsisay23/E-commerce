# Electromart 🛒

A full-stack e-commerce web application for electronics (phones, PCs, AirPods, and accessories) built using React and Laravel.

---

## 📌 Features

- Product listing and category browsing  
- Shopping cart system  
- Order placement (simulation-based checkout)  
- Admin-controlled product management  
- User authentication system  
- Order tracking structure  

---

## 🛠️ Tech Stack

**Frontend:**
- React
- HTML, CSS

**Backend:**
- Laravel (PHP)

**Database:**
- MySQL

---

## 🗄️ Database Structure

- users  
- products  
- orders  
- order_items  

---

## ⚙️ How It Works

- Users browse products and add them to cart  
- Orders are created and stored in database  
- Admin manages products and order flow  
- Authentication handled using Laravel built-in system
